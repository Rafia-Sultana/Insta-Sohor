**How does JS work?

Javascript is single threded dynamic programming language.Here are many types of system to run JavaScript on browser.Such as- chrome has V8 Engine, firefox has spider-monkey..This Engines convert javascript language to machine language.Though, it is single-threded, but it can handle multiple task at a time parallely. Because, this single-thread hand over its task to helper.Thats why,it works as a non-block programming language.


**Differnece between localStorage and sessionStorage?

We store data in localStorage and it remains in a key-value pair format.It also exist after the browser is closed and gets cleared only through javascript.It has no expiry date.But data store in sessionStorage untill the tab is closed.The data can be accessed from the same tab.
